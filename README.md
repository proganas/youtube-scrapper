# YouTube Scrapper - Laravel Assessment

This project is a Laravel-based web application that discovers educational YouTube playlists using AI-generated search queries and the YouTube Data API.

## Features

- Input multiple categories (one per line)
- Generate educational course titles using AI
- Search YouTube playlists for each generated title
- Store playlists in the database
- Prevent duplicate playlists using unique `playlist_id`
- Display results in a clean Bootstrap-based UI

## Tech Stack

- Laravel
- Bootstrap 5
- OpenAI API
- YouTube Data API v3
- MySQL
